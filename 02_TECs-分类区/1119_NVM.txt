

NVM：
****************************************************************
<<<<<<<< 


>>>>>>>>【概念】
NVM => Node Version Manager（Node版本管理器）.它可方便地在机器上安装/维护多个Node版本


>>>>>>>>【安装】
安装：https://github.com/coreybutler/nvm-windows/releases
安装办法，参考：https://segmentfault.com/a/1190000009442289


>>>>>>>>【命令】
安装Node：        nvm install x.x.x (或 latest)
使用Node：        nvm use x.x.x
移除Node：        nvm uninstall x.x.x
看版本(node)：    nvm list
看版本(nvm)：     nvm v
